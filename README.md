# 🕋 Quran Center Management System (Student Records)

### عن المشروع (Arabic Description)
نظام تقني مخصص لإدارة حلقات التحفيظ ودروس التقوية، يهدف إلى تسهيل أرشفة بيانات الطلاب ومتابعة تقدمهم في حفظ القرآن الكريم إلكترونياً.

### المهارات المستخدمة (Technologies)
- **Backend:** Python (Flask)
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **Interactive Features:** JavaScript (Live Search)

### كيفية التشغيل (How to run)
1. إنشاء قاعدة بيانات MySQL باسم `quran_center`.
2. تثبيت المكتبات: `pip install flask mysql-connector-python`.
3. تشغيل الملف: `python app.py`.

**المبرمج:** براء محمود عكاوي - طالب معلوماتية (TS)
